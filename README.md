# 🎲 Dice Game

A fun and interactive **Dice Game** built using **React, Vite, and Tailwind CSS**.  
In this game, players select a number and roll the dice to test their luck.

🔗 **Live Website**  
https://dice-game-mocha-eta.vercel.app/

## ✨ Features

- 🎯 **Number Selection** — Choose any number between 1 and 6
- 🎲 **Dice Roll** — Roll the dice to test your luck
- 📈 **Score System** — Score increases if your guess matches the dice
- 📉 **Penalty System** — Score decreases if the guess is incorrect
- 🔄 **Reset Score** — Reset the game score anytime
- 📜 **Game Rules Toggle** — Show or hide the rules section
- 📱 **Responsive Design** — Works on desktop and mobile devices

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Amitkavathekar/Dice-Game.git
```

### Navigate to the project folder

```bash
cd Dice-Game
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open in browser:

```
http://localhost:5173
```

---

## 🎮 Game Rules

1. Pick any number from **1 to 6**.
2. Click **Roll Dice**.
3. If the dice matches your selected number, your score increases by the dice number.
4. If the dice does not match, **1 point will be deducted**.
5. You can **reset the score anytime** using the reset button.
